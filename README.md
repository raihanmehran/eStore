# eStore
A web application for buying goods. Built with React and .NET Core.

## Backend
- C#
- .NET Core 7 Web APIs
- EF Core 7
- SQLite (development)
- Postgres (production)

## Frontend
- React
- Axios
- Redux (for state management)
- Material UI

## To Run the project
- Clone the project

### How to run backend:
- `cd API`
- `dotnet run`

### How to run backend:
- `cd client`
- `npm start`
